# Glycan Pathway Simulator

糖鎖の生合成を、ネットワークフローとして触って理解するためのインタラクティブなWebデモです。

このアプリでは、糖鎖が作られる過程を「線の太さ」「流れる粒子」「最終分布の変化」として可視化します。ユーザーはネットワーク上の線を選び、通しやすさを変えることで、糖鎖A / 糖鎖B / 糖鎖Cの割合がどう変わるかをリアルタイムに観察できます。

## 目的

糖鎖の生合成過程は、多数の反応がつながったネットワークとして捉えることができます。このデモは、その考え方を文章ではなく操作で理解できるようにすることを目的にしています。

基本的な流れは次の通りです。

1. 線を選ぶ
2. 通しやすさを変える
3. 流量が再配分される
4. 最終的な糖鎖分布が変わる

## 主な機能

- ネットワーク上の線を直接選択
- スライダーで各反応の通しやすさを変更
- 停止ボタンで特定の反応を止める
- 線の太さと粒子の動きで流量を表示
- 糖鎖A / 糖鎖B / 糖鎖Cの分布をリアルタイムに比較
- 30秒チャレンジで、目標割合に近づけるゲームモード
- Desktop / Tablet / Mobile に対応したレスポンシブUI

## モデルの見方

このネットワークは、研究内容を直感的に伝えるためのイメージです。実際の糖鎖構造や反応ネットワークを厳密に再現したものではありません。

デモ内では、以下の対応で生合成の流れを表現しています。

| 表示 | 意味 |
| --- | --- |
| 点や図形 | 糖鎖が作られていく途中の状態 |
| カプセル形 | 最終的に生成される糖鎖 |
| 線 | 生合成反応 |
| 線の太さ | 流量の大きさ |
| 通しやすさ | 反応の能力 |
| 停止 | 対応する反応を止めた状態 |

各ノードでは、入ってきた流れが出ていく流れへ配分されるように計算しています。線の通しやすさを変えると、下流の流量と最終分布が即座に更新されます。

## ゲームモード

ゲームモードでは、A / B / Cの目標割合が提示されます。制限時間内に線の通しやすさを調整し、現在の割合を目標に近づけます。

このモードは、反応の能力を変えると最終生成物の割合が変わることを、短時間で体験するためのものです。

## 研究との関係

実際の糖鎖ネットワークは、このデモよりはるかに大きく複雑です。手作業で条件を探しながら目的の糖鎖分布へ近づけることは現実的ではありません。

この研究では、数理最適化とコンピュータを活用し、大規模な糖鎖生成ネットワークの流れをシミュレーションする手法の開発を目指しています。実験の前にシミュレーションで有望な条件を絞り込むことで、仮説検証を加速し、メカニズム解明や治療研究への貢献につなげます。

## 検証

このリポジトリには、現時点で自動テストやビルドステップはありません。変更時は以下を確認します。

```bash
node --check app.js
git diff --check
```

加えて、主要な画面幅で表示崩れ、横スクロール、コンソールエラーがないことをブラウザで確認します。

## GitHub Pages

GitHub Pagesでは、リポジトリのルートをそのまま配信します。

1. GitHubでリポジトリを開く
2. `Settings` -> `Pages` を開く
3. `Source` を `Deploy from a branch` にする
4. Branchを `main`、フォルダを `/ (root)` にする
5. 保存する
